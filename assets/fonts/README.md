# Material Symbols Font

To use Material Symbols icons in Lectern, download the font file:

1. Visit: https://fonts.google.com/icons
2. Click "Download" to get the Material Symbols font
3. Extract the TTF file (MaterialSymbols-Regular.ttf or similar)
4. Place it in this directory as `MaterialSymbols-Regular.ttf`

Alternatively, you can download directly:
- Variable font: https://fonts.google.com/download?family=Material%20Symbols
- Or use: `wget https://github.com/google/material-design-icons/raw/master/variablefont/MaterialSymbolsOutlined%5BFILL%2CGRAD%2Copsz%2Cwght%5D.ttf -O MaterialSymbols-Regular.ttf`

The app will work without the font (using text labels), but icons will look better with the font installed.
